---

## ✨ Features

- **Pixel-Perfect UI:** Meticulously crafted to match the original Figma design, including custom typography and SVG assets.
- **Fully Responsive:** Adapts seamlessly across Desktop, Tablet (1024px), and Mobile (768px) viewports without horizontal scrolling or layout breakage.
- **Interactive Mobile Navigation:** Features a custom React-state-driven hamburger dropdown menu for smaller screens.
- **Advanced CSS Layouts:** Utilizes Flexbox for complex alignments and precise `absolute` positioning for overlapping avatars, custom quotes, and decorative background shapes.
- **Custom SVG Favicon:** Includes a lightweight, infinitely scalable custom SVG brand icon.

## 🛠️ Technologies Used

- **React.js:** Component structure and state management (`useState` for mobile navigation).
- **Vanilla CSS3:** Pure CSS for styling, avoiding external UI libraries to demonstrate core styling proficiency.
- **HTML5:** Semantic markup ensuring accessibility and clean DOM structure.

## 🚀 Getting Started

To run this project locally on your machine, follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/prerana1621/Elementum-landing-page.git
   ```

2. **Navigate to the project directory**

   ```bash
   cd Elementum-landing-page
   ```

3. **Install the dependencies**

   ```bash
   npm install
   ```

4. **Start the development server**

   ```bash
   npm run dev
   # or npm start (depending on your setup)
   ```

5. **Open in Browser**
   Open `http://localhost:5173` (or the port provided by Vite/CRA) to view it in the browser.
